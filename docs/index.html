What's inside
Everything you need, one hub.

Multi-Hub is a modular Angular dashboard built to be extended. Here's what's already running.

🤖

AI-Powered Job Search

Upload your CV and let the AI web crawler scan and match relevant job listings automatically. Filter by tags like skill, type, and location.

🎬

Movie & TV Discovery

Search any movie or TV series by title and year. Browse results showing type, release year, and jump straight to watching.

🧱

Modular Dashboard

Six dedicated hub sections — News, Movies, Application, Shop, Research, and About — with clean routing and access control per module.

🔔

Notification System

Real-time toast notifications with icon, title, and message. Integrated error handling with custom 403 access-restricted pages.

📋

Job Detail Panel

View full job details including title, location, category, salary, job type, required skills, and description — without leaving the page.

☁️

Multi-Platform Deploy

Ships to GitHub Pages (static) and Heroku (Node/Express server) with dedicated build configurations for each environment.

Modules
Hub sections at a glance

Each hub section is an independent Angular component with its own state and routing logic.

Application — Job Finder

Live

The flagship module. Connects to an AI web-crawler backend to surface personalised job matches.

  • CV file upload input
  • Dynamic tag-based filtering
  • Job cards: title, location, salary, type
  • View Details action per result
  • Angular Material raised buttons

Movies — Watch Module

Live

Search and discover movies or TV series. Powered by a REST API service with two-way data binding.

  • Search by title and year
  • Results: title, year, type (Movie / Series)
  • Watch Now button per result
  • HttpClient-based movie service
  • Reactive two-way NgModel binding
